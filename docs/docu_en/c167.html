User manual

System components

Das system consists 3 executable programs SchemaEditor, FormEditor and FormServer. There are only one starkit package but at the start of Xdobry you must choose which module do you want to start. The interfaces between these separate programs are to 2 XML-documents types; the Repository (standard file-extension .xmldbschema) and form-description (standard file-extension .xmlforms). The treatment of both XML-documents is taken over from corresponding programs, however, can take place also manually in an editor. You can find the fitting DTDs (Document Type Description) in catalogdtd.
